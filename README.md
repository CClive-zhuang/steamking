# Steam king

Steam king 一款面向 Steam 玩家，专注清单拉取与入库辅助的轻量级工具。
支持中文界面、深色主题，一键拉取 Lua/JSON 清单，辅助整理 DLC 入库脚本，让配置流程更简单，目前拥有以下支持：

-清单拉取
输入 AppID，一键拉取清单，并写入 Lua/JSON，实现本地入库功能。

-DLC 辅助整理
自动拼接 addappid 列表，方便根据需要整理 DLC 入库脚本，减少出错。

-深色界面
深色主题、基础搜索与快捷跳转，一键打开 Steam 相关页面。

（更多功能留给您自行探索。）


## 安装

首先，您的设备需要满足以下要求：

- Windows 10 1809 (17763) 及以上的版本
- 为了更好的使用体验，请在系统设置中开启**透明效果**和**动画效果**
- steam版本在2026年3月6日之前（由于steam在三月份时进行了入库在线监测，所以使用时候请进行版本回退哦！）

## 开发

在本地编译应用，你需要安装 Visual Studio 2022 并选择以下负载：

-  .NET 驱动
-  .C++ 驱动
-  .Windows10 系统安全性通用驱动
